# AGPL RiskLab · 金融資訊人員 AGPL 授權風險與防範互動學習網站

專為**金融 IT / 開發者**設計的 AGPL 授權風險互動式學習網站，亦可作為資深智財律師
**兩小時法遵講座**的投影骨架。把艱澀的授權條文，轉成看得懂、點得動的儀表板與視覺化圖表。

## 為什麼需要它

在生成式 AI 輔助寫程式的時代，開發者極易**無意間引入 AGPL 程式碼**。AGPL 的
「網路互動條款（Network Interaction, §13）」專門針對伺服器端／雲端服務——而這正是
金融系統的主場。一旦觸發，公司核心金融系統將被迫對所有網路使用者揭露原始碼。

## 快速開始

直接用瀏覽器開啟 `index.html` 即可。**零外部相依、零 CDN、可離線使用**（金融內網友善）。

```
open index.html        # macOS
xdg-open index.html    # Linux
```

## 內容區塊

1. **綜合曝險儀表板** — 半圓儀表 + KPI 卡，量化金融業的 AGPL 曝險
2. **授權光譜** — 雷達圖 / 義務熱力矩陣，定位 AGPL 在風險光譜的位置
3. **§13 觸發機制** — 流程圖拆解 GPL 與 AGPL 的唯一致命差異
4. **金融場景熱力圖** — 互動式：點選「系統類型 × 整合方式」看風險判讀
5. **AI 污染鏈** — 生成式 AI 如何把無標示的 copyleft 片段帶進核心
6. **互動決策樹** — 30 秒自評「這個元件我能用嗎」
7. **防範成熟度** — 雷達圖 + 可勾選檢核清單，建立縱深防線
8. **案例時間軸** — AGPLv3、MongoDB、Grafana 等授權變更殷鑑
9. **講座模式** — 兩小時議程表 + 可摺疊講者備忘稿
